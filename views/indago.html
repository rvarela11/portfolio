Description:

Indago helps you find the perfect day trip in the city. You can see other trips and save the best one that matches you. You can also create a custom trip from scratch.

    User Stories

  • User is able to signs up and/or login
  • User is able to select a city

  • User is able to click on any trip
  • User is able to see all the places for that trip
  • User is able to save the trip

  • User is able to click on the favorite trips tab
  • User is able to click on the favorite places tab

  • User is able to search for places
  • User is able to log out

Design Process

Research

I came up with this idea after staying at an airbnb for 2 weeks. Most of the people in the house would ask me for ideas on what to do around Austin. And most of the time I would end up saying the same thing.

Experience

The design of indango was inspired by the airbnb website. It has a lot of whitespace and subtle colors so the focus can be on the pictures and locations.

Build

This is the current stage that I am on. The application is about 60% completed.

This was my second major project and first time working with a team. My main goals where to:

  • Have a smooth UI/UX. This was done by using more white space then my previous projects.
  • Have a mobile friendly website
