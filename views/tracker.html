Description:

Tracker is a google chrome extension that helps a group of people focus on completing a learning objective. This is done by tracking how much time is spent on selected web sites. The idea is that the more time spent studying, the more likely the person will succeed in learning the material.

Technologies:

  • Angular
  • Express
  • Node.js
  • Knex
  • PostgreSQL
  • Google Chrome Extension
