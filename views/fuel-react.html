Description

Fuel helps you compare your goal calories vs your daily calories in three simple steps. The first step is to get your goal calories. This is calculated by collecting some general information. The second step is to get your daily calories. This is calculated by searching food items and placing them in a breakfast, lunch, or dinner box. The third step will give you a breakdown.

This was the second version of this application. You can find the first version here ( Code | Site ).

Overall

Main goals:

  • Provide a smooth user experience. One specific case where this is done is on the step 2 page. The option box changes views when a user clicks on the different tab buttons.
  • Provide a smooth user interface. This can be found on the goal and daily calories counter. When a user completes step 1 and/or 2, the counter is instantly updated. This is done through redux.
  • Make the application mobile friendly.

Challenge

Be able to pass the calories data for the specific item clicked to calculate the daily calories. On the first version, I passed the entire string then used regex to get the calories number to calculate the daily calories. This time I wanted to avoid using regex for it was an extra step.

Solution

To solve this, I compiled the data into an array and passed the needed object on every onClick event in the results list.

The result from the API call is passed through the renderResultList function. Two map functions run to extract the data needed. This data is placed in two different constants. An empty object named results is also created.

A map function runs through the itemsArray array which builds the data for the results object.

The results object keys are placed in a constant in order to run a map function through it. This returns an array which is stored in the foodItems constant. This array will pass the object needed on every onClick event in the results list.

Code

              renderResultList(listData) {
                const itemsArray = listData.hits.map(item => item.fields.item_name.split(" - ")[0]);
                const caloriesArray = listData.hits.map(item => item.fields.nf_calories);
                const results = {};
              
                itemsArray.map((item,index) => {
                  if(results.hasOwnProperty(item)){
                  results[item].calories.push(caloriesArray[index]);
                  } else {
                  results[item] = {};
                  results[item].calories = [];
                  results[item].calories.push(caloriesArray[index]);
                  }
                });
              
                const resultsNames = Object.keys(results);
                const foodItems = resultsNames.map((foodName,index) => {
                  return {name: foodName, calories: results[foodName].calories[0]};
                });
              
                return <div>
                  <ul>
                    {foodItems.map((item) => {
                      const displayInfo = `${item.name} = ${item.calories}`;
                        return <li key={displayInfo} onClick={()=> this.handleCheckboxChange(item)}>
                          {displayInfo}
                        </li>
                    })}
                  </ul>
                </div>
              }

More Code

This code explains how the option box changes components when a user clicks on the different tab buttons. To simplify the code, I will only be explaining the breakfast option.

A class with the name of Step2 is created with an initial state value of "Breakfast, Lunch, Dinner?".

Once the page loads, a button and the initial state value are displayed. The initial state value is rendered through the renderClickedItemDiv function.

The button has an onClick event that will run the breakfastButton function and update the initial state value. This will render a new display. 

              import React, { Component } from 'react';
              
              class Step2 extends Component {
                constructor (props) {
                  super(props);
                  this.state = {
                    resultsTitle: 'Breakfast, Lunch, Dinner?'
                  }
                }
               
                render() {
                  return <div>
                            <button onClick={this.breakfastButton.bind(this)}> Breakfast </button>
                          <div>
                            {this.renderClickedItemDiv()}
                          </div>
                        </div>
                }
              
                renderClickedItemDiv() {
                    return <div>
                            <h5> {this.state.resultsTitle} </h5>
                           </div>
                 }
              
                 breakfastButton() {
                   this.setState({resultsTitle: "Breakfast"});
                 }
              }
              
              export default Step2;
